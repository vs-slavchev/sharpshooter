sharpshooter

Minimal file manager in the terminal. Single-key hotkeys, zero friction.

Install

With pipx

pipx install git+https://github.com/vs-slavchev/sharpshooter.git

Update: pipx upgrade sharpshooter

With uv

uv tool install git+https://github.com/vs-slavchev/sharpshooter.git

Update: uv tool upgrade sharpshooter

One-liner (requires pipx or uv)

curl -fsSL https://vs-slavchev.github.io/sharpshooter/install.sh | bash

Usage

sharpshooter

Default keys

e / nup / down
k / iparent / child directory
fnew folder
rrename
c / d / pcopy / cut / paste
xdelete (to trash)
uundo delete
zzip / unzip
mmark file
htoggle hidden files
topen terminal here
?toggle hotkey guide
qquit

All keys are configurable in ~/.sharpshooter_config.
